## 4. 연금 수령 전략

### 4-1. 3층 연금 체계

```
┌─────────────────────────────────┐
│   3층: 개인연금 (연금저축/IRP)    │  ← 세액공제 활용
├─────────────────────────────────┤
│   2층: 퇴직연금 (DB/DC/IRP)      │  ← 퇴직금 운용
├─────────────────────────────────┤
│   1층: 국민연금                   │  ← 기본 소득 보장
└─────────────────────────────────┘
```

### 4-2. 수령 시기 및 방법

| 연금 종류 | 수령 개시 시기 | 수령 방법 | 절세 포인트 |
|-----------|---------------|-----------|-------------|
| 국민연금 | 만 63~65세 | 종신 월 수령 | 연기 수령 시 연 7.2% 증가 |
| 퇴직연금 (IRP) | 만 55세 이후 | 연금 수령 (10년 이상) | 연금소득세 3.3~5.5% (일시금 대비 유리) |
| 연금저축 | 만 55세 이후 | 연금 수령 (10년 이상) | 연 1,500만원 이하 분리과세 |

> [!tip] 절세 핵심
> 연금 수령 시 **연간 1,500만원 이하**로 수령하면 분리과세(3.3~5.5%)가 적용되어 세금 부담이 줄어듭니다.

## 5. 보험 점검

### 5-1. 필수 보험 체크리스트

- [ ] **실손의료보험**: 가입 여부 확인, 갱신형/비갱신형 점검
- [ ] **암보험**: 보장 내용 및 만기 확인
- [ ] **종신보험**: 필요성 재검토 (자녀 독립 후 감액/해지 고려)
- [ ] **간병보험/치매보험**: 고령기 대비 검토
- [ ] **운전자보험**: 유지 여부 확인

### 5-2. 보험료 최적화

> 퇴직 후에는 불필요한 보험을 정리하고, **실손보험 + 암보험** 중심으로 유지하는 것이 효율적입니다.

## 6. 리밸런싱 계획

### 6-1. 정기 리밸런싱

| 주기 | 점검 항목 |
|------|-----------|
| **월 1회** | 수입/지출 점검, 비상자금 확인 |
| **분기 1회** | 자산 배분 비율 점검, 목표 대비 ±5% 이상 이탈 시 조정 |
| **연 1회** | 전체 재무 계획 재검토, 세금 최적화, 보험 점검 |

### 6-2. 리밸런싱 기준

```
목표 비율 대비 ±5% 이상 이탈 시 → 원래 비율로 재조정

예시: 채권 목표 30% → 실제 36% 이상 또는 24% 이하 시 조정
```
